## Quick Deployment Steps

1. **Download this folder** to your computer
2. **Create a GitHub repository** and push these files
3. **Connect to Cloudflare Pages** via your GitHub account
4. **Add your custom domain** (gaulos.eu)
5. **Go live** — Cloudflare handles the rest

**See `CLOUDFLARE_DEPLOYMENT_GUIDE.txt` for detailed step-by-step instructions.**
